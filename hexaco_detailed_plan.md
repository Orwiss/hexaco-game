# HEXACO BHI 24문항 기반 인터랙티브 웹 성격 테스트 — 상세 기획서

## 개요

- **검사 도구**: BHI (Brief HEXACO Inventory) 24문항 (de Vries, 2013)
- **구조**: 6개 에피소드 × 에피소드당 4문항
- **세계관**: "대학원생 생존기" — 입학부터 졸업까지
- **목적**: 인터랙션이 재밌는 웹 + 학술 이론 기반 성격 테스트 + 논문 데이터 수집

---

## 1. 에피소드 구성 및 스토리

### 에피소드 1: OT 환영회 → 외향성(X) 측정

**장면 묘사**: 학기 초 OT. 강의실에 처음 보는 동기들이 모여 있다. 교수님의 간단한 인사 후 자기소개 타임이 시작되고, 이후 근처 식당에서 뒤풀이가 이어진다. 시끌벅적한 분위기 속에서 새로운 사람들과 인사를 나눈다.

**전환 내러티브**: "그렇게 정신없는 첫날이 지나고… 본격적인 대학원 생활이 시작됐다."

**측정 요인**: 외향성 (Extraversion)
- 사회적 자존감 (Social Self-Esteem): "나는 내 자신에 대해 꽤 만족한다"
- 사회적 대담성 (Social Boldness): "여러 사람 앞에서 말할 때도 별로 긴장하지 않는다"
- 사교성 (Sociability): "나는 모임이나 사교 행사를 즐긴다"
- 활력 (Liveliness): "나는 에너지가 넘치는 편이다"

**인터랙션**: 강의실 일러스트 씬. 좌석을 클릭해서 앉을 자리를 선택하면 주변 동기들이 반응한다. 문항 응답 시 키보드 키를 누르면 Patatap 스타일로 사운드+컬러 버블이 터지는 피드백. 리커트 5점은 원 크기로 시각화 (작은 원 = 매우 비동의 ~ 큰 원 = 매우 동의).

---

### 에피소드 2: 첫 수업과 과제 → 성실성(C) 측정

**장면 묘사**: 교수님이 첫 주에 과제 3개를 내줬다. 책상 위에 논문 더미, 열린 노트북, 포스트잇, 커피잔이 쌓여 있다. 데드라인이 하나씩 다가오고, 할 일 목록이 점점 길어진다.

**전환 내러티브**: "과제를 겨우 끝내고 나니… 이번엔 연구실 생활이 기다리고 있었다."

**측정 요인**: 성실성 (Conscientiousness)
- 조직성 (Organization): "나는 물건을 항상 깔끔하게 정리한다"
- 근면성 (Diligence): "나는 목표를 향해 열심히 일한다"
- 완벽주의 (Perfectionism): "나는 실수를 피하려고 여러 번 확인한다"
- 신중함 (Prudence): "나는 결정을 내리기 전에 신중하게 생각한다"

**인터랙션**: 책상 탑뷰(위에서 내려다보기). 쌓인 과제, 노트, 책 등이 어질러져 있다. 슬라이더로 응답하면 동의할수록 책상이 점점 정리되고, 비동의할수록 어질러지는 시각적 피드백. 포스트잇을 드래그해서 정리하는 마이크로 인터랙션.

---

### 에피소드 3: 연구실 생활 → 우호성(A) 측정

**장면 묘사**: 공동 프로젝트가 시작됐다. 선배가 데이터 정리를 떠넘기고, 동기와 분석 방법에 대한 의견이 갈린다. 후배가 코딩을 모르겠다며 도움을 요청한다. 연구실 안에서 다양한 관계가 교차한다.

**전환 내러티브**: "연구실에서 부대끼며 반년이 흘렀다. 어느새 첫 학회 시즌이 다가왔다."

**측정 요인**: 우호성 (Agreeableness vs. Anger)
- 용서 (Forgiveness): "나는 나에게 잘못한 사람도 쉽게 용서한다"
- 온화함 (Gentleness): "사람들이 나를 화나게 해도 침착하게 대처한다"
- 유연성 (Flexibility): "다른 사람과 의견이 다르면 타협하는 편이다"
- 인내 (Patience): "나는 다른 사람의 단점에 대해 관대하다"

**인터랙션**: 아이소메트릭 연구실 뷰. 연구실 내 동료들이 말풍선과 함께 등장한다. 문항은 부채꼴 카드로 펼쳐지고, 카드를 고르면 연구실 분위기(조명 색상, 동료 표정)가 변한다. 부드러운 선택 = 따뜻한 톤, 강한 선택 = 차가운 톤.

---

### 에피소드 4: 첫 학회 발표 → 정직-겸손(H) 측정

**장면 묘사**: 논문을 학회에 제출했다. 발표 준비 중 데이터 해석이 애매한 부분을 발견한다. 다른 연구자의 화려한 포스터를 보며 비교하게 되고, 포스터 세션에서 예상치 못한 날카로운 질문을 받는다.

**전환 내러티브**: "학회에서 돌아오니 한 학기가 또 끝나 있었다. 그리고… 번아웃이 찾아왔다."

**측정 요인**: 정직-겸손 (Honesty-Humility)
- 진실성 (Sincerity): "나는 이익을 위해 다른 사람에게 아첨하지 않는다"
- 공정성 (Fairness): "나는 규칙을 어겨서라도 이득을 취하고 싶지 않다"
- 탐욕 회피 (Greed Avoidance): "나는 사치품이나 높은 지위에 큰 관심이 없다"
- 겸손 (Modesty): "나는 특별대우를 받을 자격이 있다고 생각하지 않는다"

**인터랙션**: 학회장 횡스크롤. 포스터들 사이를 좌우로 걸어다니며 상황을 만난다. 문항을 포스터 보드에 핀으로 붙이는 인터랙션. 핀 색상이 응답 값에 따라 달라진다. 주변 관람객들이 웅성거리는 앰비언트 사운드.

---

### 에피소드 5: 번아웃 시즌 → 정서성(E) 측정

**장면 묘사**: 논문 리젝 메일이 왔다. 실험은 3번째 실패하고, 수면은 부족하다. 새벽 연구실에 혼자 앉아 모니터를 바라본다. 카톡 알림이 울리지만 답장할 기력이 없다.

**전환 내러티브**: "그 터널의 끝이 보이기 시작했다. 졸업 심사 날짜가 잡혔다."

**측정 요인**: 정서성 (Emotionality)
- 두려움 (Fearfulness): "나는 신체적 위험에 대해 공포를 느끼는 편이다"
- 불안 (Anxiety): "나는 사소한 일에도 걱정을 많이 한다"
- 의존성 (Dependence): "힘든 일이 있으면 다른 사람의 위로가 필요하다"
- 감수성 (Sentimentality): "나는 감정적으로 다른 사람에게 애착을 느낀다"

**인터랙션**: 어두운 모니터 화면 인터페이스. 화면 밝기가 응답에 따라 변한다 (불안 높음 = 어두워짐, 낮음 = 밝아짐). 타이핑 소리, 알림음, 빗소리 등 ASMR급 청각 피드백. Bruno Simon 영감의 빛/어둠 전환 연출. 마우스 커서 주변만 밝게 빛나는 효과.

---

### 에피소드 6: 졸업 심사 → 개방성(O) 측정

**장면 묘사**: 졸업 논문 심사를 앞두고 있다. 지금까지의 연구를 돌아보며, 새로운 분야에 대한 호기심이 생긴다. 심사 후 앞으로의 진로 — 학계, 산업계, 전혀 다른 길 — 의 갈림길에 선다.

**전환 내러티브**: → 결과 페이지로 전환

**측정 요인**: 개방성 (Openness to Experience)
- 미적 감상 (Aesthetic Appreciation): "나는 예술 작품을 감상하는 것을 즐긴다"
- 호기심 (Inquisitiveness): "다양한 주제에 대해 알고 싶어 한다"
- 창의성 (Creativity): "나는 독창적인 방법을 찾는 것을 좋아한다"
- 비관습성 (Unconventionality): "나는 이상하거나 독특한 사람들에게 흥미를 느낀다"

**인터랙션**: 갈림길 1인칭 시점(POV). 여러 개의 문이 있고, 각 문이 다른 미래를 상징한다. 문을 클릭하면 열리면서 안에 문항이 나타난다. 선택에 따라 배경 세계가 변하는 파티클 효과. 마지막 문항 후 모든 문이 열리며 결과 페이지로 전환.

## 3. 인터랙션 디자인 상세

### 공통 요소

**리커트 5점 척도 시각화 방식**:
- 기본: 5개의 원형 버튼 (크기가 점점 커짐)
- 선택 시 즉각적 피드백: 사운드 + 컬러 버블 + 미세 진동(haptic)
- 각 에피소드마다 시각적 테마가 다름

**전환 애니메이션**:
- 에피소드 간: 페이드 블랙 + 내레이션 텍스트 타이핑 효과
- 문항 간: 부드러운 슬라이드 또는 모프 애니메이션
- 진행률: 화면 하단에 에피소드 진행 바 (6칸)

**Patatap 영감 요소**:
- 모든 클릭/탭에 사운드 피드백
- 응답할 때마다 배경에 작은 파티클 이펙트
- 에피소드 완료 시 축하 사운드 + 시각 효과

### 에피소드별 고유 인터랙션

| 에피소드 | 뷰 타입 | 응답 방식 | 시각 피드백 | 사운드 |
|---------|---------|----------|-----------|--------|
| 1. OT | 강의실 정면 | 원형 버튼 클릭 | 컬러 버블 | 수다 소리 |
| 2. 과제 | 책상 탑뷰 | 슬라이더 드래그 | 책상 정리됨/어질러짐 | 종이 넘기기 |
| 3. 연구실 | 아이소메트릭 | 부채꼴 카드 | 조명/분위기 변화 | 키보드 타이핑 |
| 4. 학회 | 횡스크롤 | 핀 꽂기 | 핀 색상 변화 | 웅성거림 |
| 5. 번아웃 | 모니터 POV | 밝기 조절 | 화면 밝기 변화 | 빗소리/타이핑 |
| 6. 졸업 | 1인칭 갈림길 | 문 클릭 | 세계 변화 | 바람 소리 |

## 4. 결과 페이지 상세

### 채점 방식

1. 역채점 문항 반전: `반전 점수 = 6 - 원래 점수`
2. 요인별 평균 계산: 4문항의 평균 (1.0 ~ 5.0)
3. 정규화: 1~5점을 0~100%로 변환하여 레이더 차트에 표시

```
score = (raw_mean - 1) / 4 * 100
```

### 유형 분류 기준

6요인의 점수를 High(≥3.5) / Low(<3.5)로 나누고, 가장 두드러진 2~3개 요인의 조합으로 유형을 결정한다.

**8개 유형**:

| 유형 이름 | 주요 요인 패턴 | 설명 |
|----------|--------------|------|
| 🔬 은둔형 연구자 | H↑ C↑ X↓ | 묵묵히 연구에 몰입하는 타입. 자기 기준이 확고하고, 성실하게 일하지만 사교적 자리는 피한다. |
| 🎤 사교형 네트워커 | X↑ A↑ H↓ | 학회마다 인맥을 넓히는 타입. 에너지가 넘치고 사람들과 잘 어울리며, 기회를 놓치지 않는다. |
| 📋 전략적 생존자 | C↑ H↑ E↓ | 계획적으로 졸업 로드맵을 짜는 타입. 감정에 흔들리지 않고 차근차근 목표를 향해 나아간다. |
| 🔥 열정형 탐험가 | O↑ X↑ C↓ | 관심사가 넓고 에너지가 넘치지만, 마감은 항상 아슬아슬. 새로운 걸 시작하는 건 잘하는데 마무리가 약하다. |
| 🛡️ 든든한 조력자 | A↑ E↑ H↑ | 랩의 정서적 기둥. 후배를 잘 챙기고, 갈등을 중재하며, 공정함을 중시한다. |
| 💡 독립형 혁신가 | O↑ H↑ A↓ | 자기만의 연구 방향을 고수하는 타입. 창의적이고 정직하지만, 타협은 잘 안 한다. |
| 🌊 감성형 관찰자 | E↑ O↑ X↓ | 깊이 느끼고 깊이 생각하는 타입. 예민하지만 관찰력이 뛰어나고, 글 쓰는 능력이 좋다. |
| ⚡ 실행형 리더 | X↑ C↑ A↓ | 프로젝트를 끌고 나가는 추진력. 목표 지향적이고 결단력이 있지만, 가끔 동료와 부딪힌다. |

### 결과 페이지 구성

1. **레이더 차트**: 6요인 점수를 한눈에 보여줌 (D3.js 또는 Chart.js)
2. **유형 카드**: 유형 이름 + 이모지 + 한 줄 설명
3. **상세 설명**: 각 요인별 점수와 대학원 생활에서의 의미
4. **공유 카드**: 인스타 스토리/카카오톡 공유용 이미지 자동 생성 (html2canvas)
5. **면책 문구**: "이 결과는 BHI(de Vries, 2013) 간이 척도 기반이며, 전문 심리검사를 대체하지 않습니다."

### 채점 코드 (TypeScript)

```typescript
interface BHIResponse {
  [key: string]: number; // 문항별 응답 (1-5)
}

// 역채점 문항 목록 (BHI 원문 기준으로 수정 필요)
const REVERSE_ITEMS = ['x2', 'c3', 'a1', 'h2', 'e4', 'o1'];

function reverseScore(score: number): number {
  return 6 - score;
}

function calculateFactorScore(
  responses: BHIResponse,
  items: string[]
): number {
  const scores = items.map(item =>
    REVERSE_ITEMS.includes(item)
      ? reverseScore(responses[item])
      : responses[item]
  );
  return scores.reduce((a, b) => a + b, 0) / scores.length;
}

function getResultType(scores: Record<string, number>): string {
  const high = (key: string) => scores[key] >= 3.5;

  if (high('H') && high('C') && !high('X')) return '은둔형 연구자';
  if (high('X') && high('A') && !high('H')) return '사교형 네트워커';
  if (high('C') && high('H') && !high('E')) return '전략적 생존자';
  if (high('O') && high('X') && !high('C')) return '열정형 탐험가';
  if (high('A') && high('E') && high('H')) return '든든한 조력자';
  if (high('O') && high('H') && !high('A')) return '독립형 혁신가';
  if (high('E') && high('O') && !high('X')) return '감성형 관찰자';
  if (high('X') && high('C') && !high('A')) return '실행형 리더';

  // fallback: 가장 높은 요인 기반
  const maxFactor = Object.entries(scores)
    .sort(([,a], [,b]) => b - a)[0][0];
  return `${maxFactor} 중심형`;
}
```
