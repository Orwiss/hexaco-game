# HEXACO 대학원생 생존기 — 상세 기획서 v2 (구현 반영)

## 개요

- **검사 도구**: BHI (Brief HEXACO Inventory) 24문항 (de Vries, 2013) 기반 게임형 성격 측정
- **구조**: 6개 에피소드 × 에피소드당 4개 시나리오 선택
- **세계관**: "대학원생 생존기" — 입학부터 졸업까지
- **목적**: 인터랙티브 웹 게임 + HEXACO 이론 기반 성격 유형 분류 + 논문 데이터 수집
- **측정 방식**: 리커트 척도 직접 노출 없음. 시나리오 속 자유 선택이 내부적으로 1~5점으로 매핑
- **학술적 포지셔닝**: "BHI 기반 게임형 성격 측정" (원 도구가 아닌 영감 기반)

## 5. 에피소드 구성 — 시나리오 기반 인터랙션

### 공통 구조

각 에피소드는 3단계로 구성:
1. **인트로**: 에피소드 번호 + 제목 + 장면 묘사 + 시작 버튼
2. **시나리오 4개**: 상황 제시 → 5개 선택지 → 선택 시 시각 피드백 → 자동 진행
3. **전환 내러티브**: 페이드 블랙 + 타이핑 효과 → 다음 에피소드

## 6. 채점 시스템

### 역채점

BHI 원문항 기준 역채점 8개: **h2, h4, e3, x2, a4, c3, c4, o1**

단, 본 게임에서는 선택지 자체가 점수로 매핑되므로 역채점 처리가 선택지 배열에 이미 반영됨. DB 저장 시에는 원래 BHI 방향으로 정규화하여 저장.

### 요인 점수 계산

```
factorScore = 에피소드 내 4개 시나리오 점수의 평균 (1.0 ~ 5.0)
정규화 = (factorScore - 1) / 4 × 100  →  0~100%
```

### 유형 분류 — 2단계 프로세스

**1단계: 프로필 분화도 검사 (profileSD)**

```
profileSD = stddev(6개 요인 점수)
threshold = scaleRange / (2 × √numFactors) = 4 / (2 × √6) ≈ 0.816

if profileSD < 0.816 → "균형잡힌 연구자" (T9)
```

- 학술적 근거: Holland RIASEC 분화 지수, Cronbach & Gleser 프로필 분석
- 외부 정규 데이터 불필요, 척도 속성에서 수학적 유도
- 요인 수/척도 변경 시 자동 적응

**2단계: 가중 매칭 점수**

```
norm_i = factorScore_i - 3.0
typeScore = Σ(weight_i × norm_i) / Σ(|weight_i|)
```

**가중치 벡터** [H, C, X, A, E, O]:

| ID | 유형 | H | C | X | A | E | O | Σ\|w\| |
|----|------|---|---|---|---|---|---|--------|
| T1 | 은둔형 연구자 | +2 | +2 | -2 | 0 | 0 | 0 | 6 |
| T2 | 사교형 네트워커 | -2 | 0 | +2 | +2 | 0 | 0 | 6 |
| T3 | 전략적 생존자 | +2 | +2 | +1 | 0 | -2 | -1 | 8 |
| T4 | 열정형 탐험가 | 0 | -2 | +2 | 0 | 0 | +2 | 6 |
| T5 | 든든한 조력자 | +2 | 0 | 0 | +2 | +2 | 0 | 6 |
| T6 | 독립형 혁신가 | +2 | 0 | 0 | -2 | 0 | +2 | 6 |
| T7 | 감성형 관찰자 | 0 | 0 | -2 | 0 | +2 | +2 | 6 |
| T8 | 실행형 리더 | 0 | +2 | +2 | -2 | 0 | 0 | 6 |

**동점 처리**: 1위-2위 간격 < 0.1 → 두 유형 모두 표시

## 8. 결과 페이지

### 구성 (위→아래 스크롤)

1. **유형 카드**: 이모지 + 유형명 + 한 줄 설명
2. **동점 알림** (해당 시): "OO 성향도 강합니다"
3. **레이더 차트**: SVG 6축, 0~100% 정규화
4. **요인별 바 차트**: 높음(≥66.7%) / 보통 / 낮음(≤33.3%) 구간 표시
5. **상세 설명**: 유형별 3~4문장 해석 + 대학원 생활 팁
6. **비슷한 유형**: 2위 유형 카드
7. **면책 문구**: BHI 출처, 간이 척도 한계, 경향성으로 해석 안내
8. **액션 버튼**: 결과 이미지 저장 / 공유하기 / 링크 복사 / 다시 하기

### 공유

| 방법 | 구현 | SDK 필요 |
|------|------|---------|
| 이미지 다운로드 | html2canvas → PNG | 없음 |
| 공유하기 | Web Share API (navigator.share) | 없음 |
| 링크 복사 | Clipboard API | 없음 |
